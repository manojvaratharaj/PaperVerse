# 📚 PaperVerse

PaperVerse is a responsive stationery e-commerce website built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

https://manojvaratharaj.github.io/PaperVerse/

## ✨ Features

- Responsive Design
- Dark Mode
- Product Search
- Category Filter
- Shopping Cart
- Wishlist
- Mobile Navigation
- Product Hover Effects

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

## 🚀 How to Run

1. Download the project.
2. Open `index.html` in your browser.
